# Human Activity Recognition Using Smartphone Sensors

## Overview
Built an end-to-end Machine Learning pipeline to classify human activities using smartphone accelerometer and gyroscope sensor data.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Models Used
- SVM
- Random Forest
- KNN

## Results
- Achieved 99.69% accuracy
